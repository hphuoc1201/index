# House Routine

A small browser life-sim game about surviving ordinary days inside a five-floor house.

## Play

Open `index.html` in a browser. Keep the main character fed, rested, calm, and on top of laundry. There is no victory state; the game ends when exhaustion or stress goes over the limit.

## House

- Ground floor: office with four computers for work/study.
- Floor 1: kitchen for cooking and eating.
- Floor 2: bedroom for sleeping and restoring energy.
- Floor 3: sisters' room, where two sisters move around, leave the house, and can talk when home.
- Floor 4: laundry and drying area, including rain events that require collecting clothes.
